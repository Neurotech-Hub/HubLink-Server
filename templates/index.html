Hublink is a turn-key system for managing, syncing, and monitoring data from multiple node devices through gateway devices.

Nodes are any "edge" device that collect data onto an SD card. Gateways are centralized devices (one or many) used for gathering data from nodes (one or many). Gateways use hublink.cloud API endpoints to coordinate settings and data transfer to the cloud (e.g., Amazon S3).

Full Stack Solution

Hublink is the only full-stack solution for managing data from edge devices. By utilizing Bluetooth Low Energy (BLE), Hublink nodes do not rely on power-hungry and complicated WiFi environments. Standalone or grouped gateways make data transfer ultra-efficient and minimize complexity.
