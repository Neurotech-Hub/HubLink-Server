#include <Hublink.h>

const int cs = A0; // chip select pin
Hublink hublink(cs);
unsigned long lastSyncTime;

void setup()
{
    Serial.begin(9600);
    Serial.println("Hello, Hublink!");
    hublink.init("hublink"); // advertisement name
}

loop()
{
    unsigned long currentTime = millis();
    
    // try to sync every 30s
    if (currentTime - lastSyncTime >= 30000)
    {
        hublink.sync();
        lastSyncTime = millis();
    }
    
    // do your stuff here!
    
    hublink.sleep(1000);
}

How it works

Hublink is a turn-key platform for managing, syncing, and monitoring data from multiple node devices through gateway devices.

Nodes are any "edge" device that collect data onto a memory card. Gateways are centralized devices (one or many) used for gathering data from nodes (one or many). Gateways use hublink.cloud API endpoints to coordinate settings and data transfer to the cloud (e.g., Amazon S3).

Full Stack Solution

Hublink is the only full-stack solution for managing data from edge devices using Bluetooth Low Energy (BLE). Standalone or grouped gateways make data transfer ultra-efficient and give you reconfigurability on the fly from the Hublink dashboard.
