About

Our Mission

Hublink is revolutionizing data collection by seamlessly connecting memory cards to the cloud. We're building the future of scientific data management, one byte at a time.

In late 2024, a group of us were chatting in the Neurotech Hub at WashU about how to wirelessly sync data from in-cage behavior devices. Previous attempts to perform event-based syncing failed, largely because the data was complex, ever-changing, and experiment-specific.

We started to simplify the problem: what if we could just get files from the SD card on the device to the cloud with minimal interruption and impact to battery life? And that's what we've done.

Roadmap

Data Redundancy
Better versioning and cold storage.

We're working on a system that allows you to version your data and store it in cold storage. This requires better data management tools in the dashboard.

Node Management
Command each node individually.

We're working on a system that allows you to command each node individually. This will allow us to do things like update the software on the nodes or send reboot commands.

Server Efficiency
Faster load times and better analytics.

Hublink can be faster and more efficient. We are working on making the dashboard more responsive and adding more analytics tools that feel natural and easy to use.

AI Integration
Predict the data that matters to you most.

We're working on integrating AI tools into the dashboard to help with data analysis. This will allow us to do things like automatically generate plots and summaries of the data automatically based on column types and metadata.
