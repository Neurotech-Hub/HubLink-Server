Overview

Hublink is a cloud-based management system for gathering, managing, and viewing data from multiple IoT gateway devices. These gateways act as a central hub, syncing data from various spoke devices (nodes) to cloud storage. It streamlines handling data from numerous animal behavior tests, such as running wheels, and provides an intuitive dashboard for activity monitoring, settings, and device health.

Core Components

The Hublink architecture includes three primary components:

  1. Hublink Node: An embedded, Arduino-based library to mediate efficient file transfer over Bluetooth Low Energy.
  2. Hublink Gateway: A Linux-based (e.g., Raspberry Pi) deployable to find, connect, and manage file dumps from nodes.
  3. Hublink.cloud: A web-based app to sync multiple gateway settings and manage Hublink lab-wide.
