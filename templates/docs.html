Hublink: A Centralized Gateway Manager & API

Overview

Hublink is a centralized cloud-based management system that allows users to easily gather, manage, and view data from multiple IoT gateway devices. These gateways act as the central hub, syncing data from various spoke devices (nodes) to cloud storage. This system provides an efficient way to handle data from numerous animal behavior tests, such as running wheels, and presents an intuitive dashboard for monitoring activity, settings, and device health.

Features

  • Settings Configuration: Set AWS credentials, file size limits, and other configurations to manage cloud storage.
  • Data Synchronization: Ensure all experiment data is backed up to a centralized cloud storage system for accessibility.
  • Dashboard: Monitor devices, alerts, and track experiment progress in real-time.
  • REST API: Programmatically check file existence and output account settings in JSON format.

Endpoint Summary

Below is a high-level summary of the available endpoints in the Hublink system:

  • Account Dashboard: View detailed information about a specific account's activity. (/<account_url>)
  • Account Settings: View and update settings for each account. (/<account_url>/settings)
  • Data Management: View collected data and generate download links. (/<account_url>/data)
  • File Existence API: Check if specific files exist in the cloud. (/<account_url>/files)
  • Settings JSON API: Get account settings in JSON format. (/<account_url>.json)
